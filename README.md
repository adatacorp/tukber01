# ğŸ“– Tukber <a name="about-project"></a>

**Tukber** This is an online transport ride application. Inspired by Uber ride app, this aims to target tuktuk's.

## ğŸ›  Built With <a name="built-with"></a>
 
 -React Native
 -Python 

### Tech Stack <a name="tech-stack"></a>

- React
- Python 
- MySql

### Key Features <a name="key-features"></a>

- Match passengers and drivers while linking locations realtime.
- Allow realtime sharing of locations from users for safety.
- Solo rides and mixed rides.
