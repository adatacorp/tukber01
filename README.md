# 📖 Tukber <a name="about-project"></a>

**Tukber** This is an online transport ride application. Inspired by Uber ride app, this aims to target tuktuk's.

## 🛠 Built With <a name="built-with"></a>
 
 -React Native
 -Python 

### Tech Stack <a name="tech-stack"></a>

- React
- Python 
- MySql

### Key Features <a name="key-features"></a>

- Match passengers and drivers while linking locations realtime.
- Allow realtime sharing of locations from users for safety.
- Solo rides and mixed rides.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

-VScode 
-Android Device

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/adatacorp/tukber01.git
```

### Install

Install this project with:

```sh
  expo
```

### Usage

To run the project, execute the following command in your terminal:


```sh
  npx expo start
```
